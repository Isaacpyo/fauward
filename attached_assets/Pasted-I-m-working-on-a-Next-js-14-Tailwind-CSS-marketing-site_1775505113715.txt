I'm working on a Next.js 14 + Tailwind CSS marketing site called Fauward 
(a white-label logistics SaaS platform). The repo is a monorepo — the 
frontend lives at apps/frontend/src/.

Brand identity:
- Primary accent: amber-600 (#d97706), hover amber-700
- Navy: bg-[#0d1f3c] (used as dark backgrounds, brand-navy class)
- Backgrounds: white, gray-50, gray-100
- Typography: system font stack, bold headings
- Tone: professional B2B SaaS, not playful
